Animations

There are some animations pre built into the template. You can make alterations to these to make your presentation more personal.

 

  • Firstly select the text or text box you would like to animate
  • Click on the animation tab in the ribbon
  • Click on the add animation button
  • Select one of the options
  • You can also edit some of the settings for the animation by opening the animation pane and clicking on the drop down arrow next to the animation you are editing. For example you might want to change the speed or timings

When adding animations remember to ensure that these add to or emphasise the message you are trying to convey. Some times less is after all.
